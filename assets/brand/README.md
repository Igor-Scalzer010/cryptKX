# cryptKX Brand Assets

This directory contains the primary vector brand assets for cryptKX.

## Files

- `cryptkx-logo-horizontal.svg`: horizontal logo for the README, splash screens, presentations, and wide layouts.
- `cryptkx-icon.svg`: square app icon base artwork for launchers, installers, shortcuts, and favicon generation.

## Visual direction

The mark combines security and cryptography references:

- shield silhouette for protection;
- lock structure for password safety;
- circuit traces for software/security tooling;
- `KX` monogram for compact brand recognition.

## Palette

| Token | Hex | Usage |
| --- | --- | --- |
| Midnight | `#08111F` | Main dark surface |
| Deep Navy | `#0C1A2E` | Background gradient |
| Cyan | `#00E5FF` | Primary crypto accent |
| Mint | `#18F2B2` | Secondary accent |
| Violet | `#7C3AED` | Gradient depth |
| Ice | `#EAFBFF` | Light foreground details |

## Notes

The SVG files use system font fallbacks for the horizontal wordmark and do not embed external font files. If raster files are needed later, export from these SVG sources so the generated PNG/ICO assets stay consistent.
